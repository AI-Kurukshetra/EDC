# Handoff Template

Use this template to refresh `PARENT_THREAD.md` before ending a substantial session.

```md
## Active Objective

- [single current goal]

## Current State

- [what is already true]
- [what was verified]

## Current Blockers

- [exact failing file + cause]
- [external blocker, if any]

## Exact Next Steps

1. [next concrete fix]
2. [next command]
3. [next validation step]

## Supabase Status

- [auth connected / blocked]
```

Rules:

- Keep bullets short.
- Prefer exact file paths and commands.
- Remove stale bullets instead of appending logs.
- Do not paste long command output into `PARENT_THREAD.md`.
